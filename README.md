
# Smart Plantation Tracking & AI Chatbot System

A full-stack MERN application for plantation management with AI chatbot support in Telugu and English.

---

# Features

## Plantation Management
- Add plants
- Update plant details
- Delete plants
- Track plant growth
- Monitor plant health

---

## AI Chatbot
- English support
- Telugu support
- Plant information assistant
- Smart plant search
- Voice input support

---

## Plant Search
Search plants using:
- Plant name
- Telugu name
- Plant ID

Example:
- Mango
- మామిడి
- Rose
- రోజా

---

## 🔐 Authentication
- User registration
- User login
- Admin dashboard

---

## QR Code Support
Each plant can have:
- Unique QR code
- Quick plant access
